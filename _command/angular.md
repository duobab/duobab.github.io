### nrm
- npm install -g nrm
- nrm ls
- nrm current
- nrm use npm源
- nrm add npm源名称 npm源url
- nrm del npm源名称
- nrm test npm源名称

### ng
- ng new 自定义项目名称
- ng generate  application 自定义应用名称
- ng generate component 自定义组件名称
- ng generate service 自定义服务名称
