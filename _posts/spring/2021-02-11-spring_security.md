## 1.1 简介
目前主流的权限管理框架有Shiro 和 Spring Security。  
Shiro是 Java 领域老牌的权限管理框架, 它轻量、简单、易于集成。  
Spring Security 最初是叫Acegi Security, 重量级、配置繁琐、门槛高, 直到 Spring Boot的出现, 这些问题统统都得到缓解。  
权限管理框架最核心的功能：认证+授权。
